# ✦ STARDUST 星塵小站

一个二次元风格的现代个人博客模板，纯 HTML/CSS/JS 实现，无需构建工具。

## ✨ 特性

- 🌙 **暗色/亮色主题** — 一键切换，跟随系统偏好
- 🎨 **极光动态背景** — 渐变光带 + 星空闪烁 + 花瓣飘落
- 💬 **弹幕式说说** — 发布的说说自动在全站飘动
- 📝 **文章分类过滤** — 技术/生活/动漫/设计/随笔/旅行
- 👥 **友链展示** — 精美的友链卡片
- ⚡ **加载动画** — 首次访问展示旋转星轨动画
- 📱 **响应式设计** — 完美适配手机/平板/桌面
- 🔌 **零依赖** — SVG 图标内嵌，无外部图标库依赖
- 💾 **localStorage** — 说说不丢失，刷新还在

## 🚀 使用方法

1. 克隆或下载本仓库
2. 用任意浏览器打开 log/index.html
3. 编辑 HTML 文件中的示例内容替换为自己的信息

## 📁 文件结构

`
blog/
├── index.html      # 首页
├── posts.html      # 文章列表
├── chatters.html   # 说说（弹幕）
├── friends.html    # 友链
├── about.html      # 关于我
├── style.css       # 样式表
└── app.js          # 交互脚本
`

## 🎨 自定义

- 修改 style.css 中的 CSS 变量来更换主题色
- 修改各 HTML 中的示例数据
- 添加真实文章链接替换 # 占位符
- 在 bout.html 中更新个人信息
